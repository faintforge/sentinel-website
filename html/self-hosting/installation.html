Installing Sentinel

Prepping Sentinel

The Sentinel repository is hosted on GitHub. In order to download it locally to your VPS run:

$ git clone https://github.com/faintforge/sentinel.git

Now a directory named 'sentinel' should appear in the same directory you ran the command in.

Enter into the directory and run:

$ python main.py

This should start Sentinel in debug mode. This is NOT meant to be used for continual use.

If no errors are thrown then you can go on to the next step.

Configuring NGINX

NGINX will be our reverse proxy. To configure it, all you need to do is move the nginx.conf file from the sentinel directory to /etc/nginx/

$ sudo cp ./nginx.conf /etc/nginx/

Some edits need to be made to the configuration file. Change all instances of WEBSITE_URL to your website url.
