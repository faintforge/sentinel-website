Setting up a VPS

Aquire a VPS

Firstly, you must aquire a VPS that allows you to run a reverse proxy, such as Apache or NGINX. This tutorial will be using NGINX.

Hostinger has been tested and confirmed to work with Sentinel, though there's no affiliation between the two.

Install dependencies

After connecting to your VPS, probably visa SSH, run the following command to make sure all dependencies are installed.

$ sudo apt install python3 nginx gunicorn

This command is used for Debian based operating systems.
